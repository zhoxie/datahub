What is Data Hub?

"Data ecosystems are diverse too diverse. DataHub's extensible metadata platform enables data discovery, data observability and federated governance that helps you tame this complexity. DataHub's extensible metadata platform enables data discovery, data observability and federated governance that helps you tame this complexity. "
-- From Datahub Official website.

Datahub.webex.com, which based on the open source Datahub project, is a metadata management platform built for Webex services. You can find various kinds of metadata describing various datasets or pipelines created by Webex services, including but not limited to dataset schema, lineage, pipeline definition, data owner, data lifecyle etc. With datahub.webex.com, you can quickly find the data you want and know more about it.

Datasets

Manage the dataset schema and the lineage of each dataset for various data platform, including Kafka, Hive, TiDB, ES, Pinot, Postgresql, Oracle, etc. In this module gives you a deeper insight into the data

Pipelines

Manage the pipeline schema and the lineage of each dataset and pipeline, you can easily know how data be generated and processed in your org.

Dashboards

Manage the dashboards and charts in different visualization platforms, with this module you can easily find the dashboard you want and you'll have a better understanding of the dashboard in your org.

Datasources

Manage the datasource information in your org. This module works as an centralized data source configuration services, you can get the data source information you need with the correct permissions.
